<h1 align="center">Speech To Text App</h1>

This is a simple Speech to Text web application developed using JavaScript. It allows users to convert spoken words into written text using the Web Speech API.

## Features

- **Speech Recognition**: Utilizes the Web Speech API to recognize speech from the user's microphone.
- **Real-time Conversion**: Transcribes spoken words into text in real-time.
- **Easy to Use Interface**: Provides a simple and intuitive user interface for speech input and text output.

## Usage
1. Allow microphone access when prompted.
2. Click the "Start Listening" button to begin speech recognition.
3. Start speaking into your microphone.
4. Your speech will be transcribed into text in real-time.
5. Click the "Stop Listening" button to end speech recognition.
6. Copy the data using "Download" button.

## Technologies Used
- HTML: Provides the structure of the web page.
- CSS: Adds styling to the web page.
- JavaScript: Implements the functionality for speech recognition using the Web Speech API.
